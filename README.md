# NO Regex

> A lot of regex functions so you don't have to worry about it

## Install

`npm install no-regex`

## How to use

```javascript
import { isEmail } from "no-regex";
```

## Methods

### Validations

> isEmail

```javascript
isEmail("fernando@email.com"); // true
```
